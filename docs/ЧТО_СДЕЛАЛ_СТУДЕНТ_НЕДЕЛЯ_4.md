# Что сделал студент на Неделе 4

Краткий перечень в формате «что сделал — где это видно». Удобно использовать при объяснении работы наставнику.

## 1. Автоматизация запуска

**Сделал:** Настроил запуск PostgreSQL и Spring Boot в контейнерах и скрипты для одной команды.

- **PostgreSQL в контейнере** — в файле **`docker-compose.yml`** описал сервис `postgres` (образ, база, пароль, порт, volume, healthcheck).
- **Spring Boot в контейнере** — в **`docker-compose.yml`** сервис `backend`, в **`backend/Dockerfile`** двухэтапная сборка: сначала Maven собирает JAR, потом образ только с JRE и этим JAR.
- **Связь приложения с БД** — в `docker-compose.yml` у `backend` указал переменные `SPRING_DATASOURCE_URL=jdbc:postgresql://postgres:5432/...` и `depends_on: postgres`, в **`backend/src/main/resources/application.yml`** — подключение к БД через переменные окружения.
- **Развёртывание БД при старте без повтора** — в `application.yml` включил Liquibase (`spring.liquibase.change-log`), в **`backend/src/main/resources/db/changelog/`** добавил master-файл и **`changelog/001-create-tables.xml`** с созданием таблиц. Liquibase сам не выполняет уже применённые изменения.
- **Скрипты запуска:** **`scripts/start.bat`** (Windows) и **`scripts/start.sh`** (Linux) — проверка Docker и вызов `docker compose up -d`. Листинг — в **`docs/SCRIPTS.md`**.
